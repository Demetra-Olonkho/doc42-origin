или

с полями вроде:

Тогда для организационного обсуждения вторым сообщением после постановки задачи мог бы идти примерно такой документ:

```text
Паспорт темы

Организационное положение проекта Деметра Олонхо

Происхождение:
Проект Деметра Олонхо

- Publishing Pipeline
- VTT
- Орден Хоронуса
- Lynxy Publishing

Цель:
Определение организационного положения проекта

Результат:
Не определён.

Статус:
Обсуждение открыто.
```
