# DEM-0001 — Терминологическая и символическая архитектура

Версия: 1.0

Статус: Accepted

Дата принятия: TBD

# 1. Назначение документа

Настоящий документ определяет терминологические, символические и номенклатурные принципы проекта Деметра Олонхо (Demetra Olonkho).

Цель документа — обеспечить единообразие именования компонентов проекта, сохранить смысловую целостность архитектуры и зафиксировать культурные и концептуальные основания выбранной номенклатуры.

Документ относится к числу основополагающих документов проекта.

---

# 2. Общие положения

Проект использует осмысленную систему именования.

Наименования компонентов проекта рассматриваются не как произвольные технические идентификаторы, а как часть архитектурного языка системы.

Имена должны помогать пониманию назначения компонента и отражать его роль в общей структуре проекта.

---

# 3. Принцип семантического именования

При выборе имени предпочтение отдаётся именам, обладающим самостоятельным смысловым содержанием.

Предпочтительными источниками являются:

- исторические личности;

- мифологические персонажи;

- культурные традиции;

- литературное наследие;

- фольклорные образы.

Название должно отражать функцию компонента, а не особенности его реализации.

Предпочтительная последовательность выбора:

```
Смысл
    ↓
Назначение
    ↓
Имя
```

Использование исключительно технических названий не рекомендуется, если существует возможность подобрать содержательное имя.

# 5. Резерв именования

Проект допускает формирование пула зарезервированных имён для будущих компонентов.

Наличие имени в резерве не означает обязательного появления соответствующего компонента.

---

## 5.1. Резерв первого уровня

Имена, имеющие прямую смысловую связь с уже существующими компонентами.

Текущий перечень:

- Ирис (Iris);

- Эйрена (Eirene).

---

## 5.2. Резерв второго уровня

Имена, представляющие потенциальный интерес для будущих компонентов.

Текущий перечень остаётся открытым.

---

# 6. Культурная политика именования

Проект не ограничивается какой-либо одной культурной традицией.

Допускается использование имён и образов из различных культурных источников.

Предпочтение отдаётся:

- культурному разнообразию;

- уважительному использованию источников;

- сохранению первоначального значения имени.

Использование имён должно способствовать расширению культурного кругозора участников проекта.

---

# 7. Принцип внутренней согласованности

Новые имена должны быть совместимы с уже существующей системой именования.

Предпочтительно, чтобы новое имя:

- обладало собственным смыслом;

- отражало назначение компонента;

- не противоречило существующей символической архитектуре;

- было пригодно для международного использования.

---

# 8. Правила документирования имён

В русскоязычной документации первое упоминание имени оформляется следующим образом:

```
Русское имя (English Name)
```

Примеры:

```
Деметра (Demetra)
Олонхо (Olonkho)
Ирида (Irida)
Ирина (Irina)
```

При последующих упоминаниях допускается использование только русской формы.

В англоязычной документации используется официальная английская форма имени.
